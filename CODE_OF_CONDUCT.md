# Code of Conduct

ViralMint follows the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**. By participating in this project — issues, pull requests, discussions, or any other channel — you agree to abide by it.

## TL;DR

- Treat people with respect. Disagree with ideas, not with people.
- No harassment, personal attacks, or discriminatory language.
- Assume good intent. Maintainers are volunteers.
- If something feels off, report it (see below).

## Reporting

If you witness or experience behavior that violates the Code of Conduct, please report it privately so we can act on it without exposing the reporter:

- Open a [private security advisory](https://github.com/openclaw-easy/ViralMint/security/advisories/new) on GitHub (yes, the same channel as security reports — it's the only built-in private channel a public repo has). Title it `[CoC]` so we route it correctly.

Reports are confidential. We will respond within 7 days and act in proportion to the severity of the violation, up to and including a permanent ban from the project.

## Scope

The Code of Conduct applies to all project spaces — the GitHub repo, issues, pull requests, discussions, and any community space that identifies itself with ViralMint (Discord, etc., should we add them). It also applies when an individual is officially representing the project in public.
